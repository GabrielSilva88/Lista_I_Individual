Sobre JavaScript

JavaScript é uma linguagem de programação interpretada estruturada

Abreviado como JS e um script em alto nível com tipagem dinâmica fraca e multi-paradigma (protótipos, orientado a objeto, imperativo e, funcional)

Sobre Bootstrap 4

É um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript,

Baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Web Desing

É uma extensão da prática do design gráfico, onde o foco do projeto é a criação de web sites e documentos disponíveis no ambiente da World Wide Web

O web design tende à multidisciplinaridade, uma vez que a construção de páginas web requer subsídios de diversas áreas técnicas, além do design propriamente dito
